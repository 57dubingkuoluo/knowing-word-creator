# 儿童识字小报生成器

根据预置主题或自定义输入，生成儿童识字小报的 AI 图像生成工具。

## 功能特点

- 5 套预置主题（超市、医院、公园、学校、餐厅）
- 支持自定义主题和标题
- 自动联想相关词汇并填充提示词模板
- 一键生成儿童识字小报图片

## 使用方法

1. 用浏览器打开 `index.html`
2. 输入并保存 API Key（Nano Banana Pro）
3. 选择预置主题或勾选"自定义主题"
4. 点击"生成图片"
5. 等待生成完成，查看结果

## 技术栈

- 原生 HTML + CSS + JavaScript
- Nano Banana Pro API

## 文件结构

```
├── index.html      # 主页面
├── css/
│   └── style.css  # 样式
└── js/
    ├── app.js     # 主逻辑
    ├── api.js     # API 封装
    └── presets.js # 预置主题数据
```
